About Kids Word Practice

What is Kids Word Practice?

Kids Word Practice is a free, open-source educational tool designed to help young children learn and review English vocabulary in a fun, interactive way. It is created for parents and teachers to support children's English learning journey.

Key Features

  • Level-based learning (P1/P2) to match different learning stages
  • Custom word lists from Excel (.xlsx) files
  • Day-by-day word practice for systematic learning
  • Audio pronunciation (3 times repetition) for better pronunciation
  • Previous/Next navigation for flexible practice
  • Hidden word feature to test memory and comprehension
  • 100% free, no ads, no data collection

How to Use

  1. Select a level (P1 for primary school 1, P2 for primary school 2)
  2. Choose a word list file (e.g., P1T1_BQ1)
  3. Enter a day number to filter specific words
  4. Practice steps:
    • Read the Chinese meaning first
    • Click "Show Word" to reveal the English word
    • Click "Read 3x" to listen to the pronunciation
    • Use Previous/Next to switch between words

Open Source

This project is completely open source and hosted on GitHub. You can:

  • View the source code
  • Report bugs or suggest improvements
  • Customize the tool for your own use

GitHub Repository: https://github.com/vizaiweb/word-review

Our Mission

We aim to provide a simple, effective, and privacy-friendly tool to make English vocabulary learning enjoyable for kids, without any cost or data collection.
